# SpendWise – Expense & Subscription Manager

SpendWise is a full-stack web application designed to help users track personal expenses and recurring subscriptions in a structured and scalable manner. The project focuses on clean data modeling, secure authentication, and a responsive user interface.

## 🚀 Features
- User authentication and authorization
- Expense tracking with categories and dates
- Recurring subscription management
- Responsive dashboard for expense overview
- RESTful API architecture

## 🛠 Tech Stack
- Frontend: React, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## 🎯 Project Goal
The goal of SpendWise is to build a reliable, analytics-ready full-stack foundation that mirrors real-world financial applications and can be extended with data analytics and insights in the future.

## 📌 Note
This project focuses on core full-stack engineering and system design and does not provide financial advice.
